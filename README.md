Banco Imobiliário Online 🎲

Versão digital inspirada no clássico Banco Imobiliário, desenvolvida para jogar diretamente no navegador com amigos em tempo real. Não é necessário instalar nada: basta criar uma sala, compartilhar o link e começar a partida.

Principais Recursos
🌐 Multiplayer online em tempo real
🔗 Convite rápido através de link compartilhável
🎲 Sistema completo de turnos e rolagem de dados
🏠 Compra, venda e gerenciamento de propriedades
💰 Controle automático de saldo, aluguel e transações
👥 Suporte para múltiplos jogadores na mesma partida
📱 Interface responsiva para computador, tablet e celular
⚡ Sem necessidade de cadastro para jogar
🎮 Criação instantânea de salas privadas
Como Funciona
Crie uma nova partida.
Compartilhe o link da sala com seus amigos.
Todos entram pelo navegador.
A partida sincroniza automaticamente em tempo real.
Divirtam-se disputando o império imobiliário!
Tecnologias
Frontend moderno baseado em JavaScript/TypeScript
Comunicação em tempo real via WebSockets
Interface responsiva e otimizada para dispositivos móveis
Arquitetura preparada para partidas multiplayer online
Objetivo do Jogo

Compre propriedades, construa seu patrimônio, cobre aluguéis dos adversários e leve seus concorrentes à falência para se tornar o maior magnata imobiliário da mesa.

🚀 Crie uma sala, envie o link para seus amigos e jogue de qualquer lugar do mundo.
