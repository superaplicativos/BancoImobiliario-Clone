# BancoImobiliario-Clone
Banco Imobiliário Online é um jogo multiplayer em tempo real inspirado no clássico de tabuleiro. Crie uma sala, compartilhe o link e jogue com amigos diretamente pelo navegador, sem instalação. Possui sistema de propriedades, transações automáticas, turnos sincronizados e suporte para múltiplos jogadores.
